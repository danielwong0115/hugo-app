Building My Portfolio with Hugo#

The Project#

Creating a personal portfolio is an essential step for any developer. It’s a place to showcase your work, share your thoughts, and connect with others in the tech community.

Why a Static Site?#

Static sites offer several advantages:

Performance Benefits#

  • Fast loading times - No server-side processing required
  • Security - Fewer attack vectors compared to dynamic sites
  • Cost-effective - Can be hosted for free on platforms like GitHub Pages

Developer Experience#

The workflow is incredibly smooth:

  1. Write content in Markdown
  2. Preview locally with Hugo’s dev server
  3. Push to GitHub
  4. Automatic deployment via GitHub Actions

Design Choices#

For this portfolio, I chose the PaperMod theme because of its:

  • Clean and minimalist design
  • Excellent readability
  • Built-in features like dark mode support
  • Responsive layout

You can learn more about PaperMod on its GitHub repository.
